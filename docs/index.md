# B Corp Navigator — Your AI Team for B Corp Certification

**Navigate 1,245 pages of B Lab Standards with 10 specialised AI skills.**

This library gives you **free, ready-to-use AI skills** that guide your company through B Corp certification and ongoing reporting. Each skill covers one impact area — from governance and fair work to climate action and compliance — and works across **Kimi Code CLI**, **Claude Code** and **Codex CLI**.

No coding required. Pick your assistant, download the matching ZIP, and start working through your B Corp roadmap with a single command.

[Meet the 10 skills](#the-10-skills) · [How to install](#how-to-install-a-skill-in-30-seconds) · [Get the repo](https://github.com/hartmut-ux/ai-bcorp-team-en)

## What is the B Corp Navigator?

The B Corp Navigator is a virtual team of AI skills built on the **B Lab Standards V2.2**. Instead of reading hundreds of pages of requirements, you install a skill and ask it questions like:

- "What do we need for Foundation Requirement 1?"
- "Draft a stakeholder governance policy."
- "Create a climate transition plan for Year 3."
- "Map our impact report to GRI and ESRS."

Each skill contains the relevant standard, templates, and a step-by-step workflow.

## How it works

1. **Start with the Certification Navigator** to assess eligibility and build your roadmap.
2. **Work through the impact-topic skills** one by one (Governance, Fair Work, JEDI, Human Rights, Climate, Environment, Public Affairs).
3. **Use the Impact Reporter** to draft annual reports mapped to GRI/ESRS.
4. **Run the Compliance Monitor** for gap analysis and audit preparation.

## How to install a skill in 30 seconds

### 1. Download the ZIP

Pick a skill from the table below and download the version for your assistant:

- **`-all.zip`** if you use more than one assistant
- **`-kimi.zip`** for Kimi Code CLI
- **`-claude.zip`** for Claude Code
- **`-codex.zip`** for Codex CLI

### 2. Unzip and copy one folder

**Kimi Code CLI:**

```bash
cp -R .kimi/skills/<skill-name> ~/.kimi/skills/
```

Then type in Kimi: `/<skill-name>`

**Claude Code:**

```bash
cp -R .claude/skills/<skill-name> ~/.claude/skills/
```

Then type in Claude: `<skill-name>`

**Codex CLI:**

```bash
cp -R .codex/skills/<skill-name> ~/.codex/skills/
```

Then type in Codex: `<skill-name>`

## Frequently asked questions

### What is an AI skill?
An AI skill is a reusable instruction pack that teaches your AI assistant how to perform a specific task with consistent quality and fewer prompts.

### Do these skills work with Kimi, Claude and Codex?
Yes. Every skill has wrappers for Kimi Code CLI, Claude Code and Codex CLI.

### Are these skills based on the current B Lab Standards?
Yes. They are built on **B Lab Standards V2.2 (February 2026)**.

### Do I need to pay?
No. All skills are released under the MIT License.

### Can I use the same skill on multiple assistants?
Yes. The `-all.zip` file contains wrappers for Kimi, Claude and Codex.

### Do I need to code?
No. You download a ZIP, copy one folder, and invoke the skill by name.
